Roadmap Operacional Ethernum

Este documento organiza as proximas fases do site sem implementar os sistemas grandes agora. A linha principal continua estatica, publicada via GitHub Pages, enquanto a estrutura e preparada para uma futura modularizacao.

Diretriz: documentar, preparar e manter o deploy estavel. Missoes, runas, busca global, export/import e painel 2.0 ficam apenas planejados nesta fase.
